Diferentierings regel

d f(x) g(x) dx ⁢ = ⁢ d f(x) dx g(x) ⁢ + ⁢ f(x) d g(x) dx

Hvis ⁢ ⁢ f(x) ⁢ = ⁢ x ⁢ ⁢ og g(x) = ⁢ x n - 1

d x n dx ⁢ = ⁢ dx dx x n - 1 ⁢ + ⁢ x d x n - 1 dx ⁢ = ⁢ x n - 1 ⁢ + ⁢ x d x n - 1 dx

Dermed må d x n - 1 dx ⁢ = ⁢ x n - 2 ⁢ + ⁢ x d x n - 2 dx

Så

d x n dx ⁢ = ⁢ x n - 1 ⁢ + ⁢ x ( x n - 2 ⁢ + ⁢ x d x n - 2 dx ) ⁢ = ⁢ x n - 1 ⁢ + ⁢ x ( x n - 2 ⁢ + ⁢ x ( x n - 3 ⁢ + ⁢ x ( x n - 4 ⁢ + ⁢ x d x n - 4 dx ))) ⁢ = ⁢ 4 x n - 1 ⁢ + ⁢ x 4 d x n - 4 dx ⁢ = ⁢ m x n - 1 ⁢ + ⁢ x m d x n - m dx

Så for m = n - 1 (m = n giver det samme resultat)

d x n dx ⁢ = ⁢ (n - 1) x n - 1 ⁢ + ⁢ x n - 1 dx dx ⁢ = ⁢ n x n - 1

Integralet af x fra 0 til t er lig arealet for en retvinklet trekant med begge katete sidernes længde lig t

Partiel integration (Integration by parts) er givet ved følgende formel

∫ f(x) g(x) dx ⁢ = ⁢ f(x) ∫ g(x) dx ⁢ - ⁢ ∫ d f(x) dx ( ∫ g(x) dx ) ⁢ dx

Hvis man sætter:

f(x) g(x) ⁢ = ⁢ x n , ⁢ ⁢ f(x) ⁢ = ⁢ x n - 1 ⁢ ⁢ and g(x) = ⁢ x

Så fåes

∫ f(x) g(x) dx ⁢ = ⁢ x n - 1 ∫ x dx ⁢ - ⁢ ∫ d x n - 1 dx ( ∫ x dx ) ⁢ dx ⁢ = ⁢ x n - 1 1 2 x 2 ⁢ - ⁢ ∫ d x n - 1 dx 1 2 x 2 ⁢ dx ⁢ = ⁢ 1 2 x n + 1 ⁢ - ⁢ ∫ 1 2 x 2 d x n - 1 dx ⁢ dx

Hvis vi udnytter at ⁢ ⁢ d x n - 1 dx ⁢ = ⁢ (n - 1) x n - 2

∫ x n ⁢ dx ⁢ = ⁢ 1 2 x n + 1 ⁢ - ⁢ ∫ 1 2 x 2 (n - 1) x n - 2 ⁢ dx ⁢ = ⁢ 1 2 x n + 1 ⁢ - ⁢ 1 2 (n - 1) ∫ x n ⁢ dx

Så isolere vi ∫ x n ⁢ dx på venstresiden.

∫ x n ⁢ dx ⁢ + ⁢ 1 2 (n - 1) ∫ x n ⁢ dx ⁢ = ⁢ 1 2 x n + 1

∫ x n ⁢ dx (1 ⁢ + ⁢ 1 2 (n - 1) ) ⁢ = ⁢ 1 2 x n + 1

∫ x n ⁢ dx ⁢ = ⁢ 1 2 1 (n + 1) 1 2 x n + 1 ⁢ = ⁢ 1 n + 1 x n + 1

Hvis vi bruger partiel integration igen på sidste led med ⁢ ⁢ f(x) ⁢ = ⁢ 1 2 x 2 ⁢ ⁢ and g(x) = ⁢ d x n - 1 dx

∫ f(x) g(x) dx ⁢ = ⁢ f(x) ∫ g(x) dx ⁢ - ⁢ ∫ d f(x) dx ( ∫ g(x) dx ) ⁢ dx ⁢ = ⁢ 1 2 x 2 ∫ d x n - 1 dx dx ⁢ - ⁢ ∫ d 1 2 x 2 dx ( ∫ d x n - 1 dx dx ) ⁢ dx ⁢ = ⁢ 1 2 x 2 x n-1 ⁢ - ⁢ ∫ d 1 2 x 2 dx ( x n - 1 ) ⁢ dx
