1 inch is equal 2,54 cm
The screen ratio is eithe 16:9 or 21:9
That meens that Width (w) divided by Height (h) is eithe 16/9 or 21/9 (ratio)
And w = ratio * h
We also has for the square of the diagonal (d) is d2 = w2 + h2 (Pythagoras)
d2 = w2 + h2 = ratio2 * h2 + h2 = h2 (ratio2 + 1)
or d2 = w2 + h2 = w2 + (w/ratio)2 = w2 (1 + (1/ratio)2)
So h2 = d2 / (ratio2 + 1)
And w2 = d2 * (1/((1/ratio)2 + 1)) = ratio2 * d2 /(1 + ratio2) = ratio2 * h2
